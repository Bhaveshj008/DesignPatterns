# Design Patterns

An interactive visual simulator for learning design patterns.

## Overview

PatternLabs provides simulations for design patterns across three categories:
- Creational Patterns
- Structural Patterns
- Behavioral Patterns

## Patterns

### Creational
Singleton, Factory Method, Abstract Factory Builder, Prototype

### Structural
Adapter, Bridge, Composite, Decorator, Facade, Flyweight, Proxy

### Behavioral
Chain of Responsibilty, Command, Iterator, Mediator, Memento, Observer, Strategy, State, Template Method, Visitor

## Tech Stack

React, TypeScript, Vite, Tailwind CSS, Lucide Icons
